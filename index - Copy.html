Project 1 Details

Problem Statement

In this study, a set of state-of-the-art (SOTA) neural network quantization techniques were evaluated by experimenting with them on different SOTA neural network model architectures. The goal was to investigate the applicability of the considered quantization techniques for production use cases, specifically image classification and object detection, in order to achieve size reduction while retaining the performance of the models.

Actions

To achieve the goals of the study, several models were trained for image classification and object detection using different datasets. These models were then quantized using the frameworks TensorFlow Lite, QKeras, and SQuant. The quantization techniques were then evaluated based on their ability to achieve model compression while maintaining accuracy.

Results

Based on the experiments, a set of recommendations were delivered for applying quantization in production. The recommendations aim to achieve less than a 2% drop in accuracy while achieving up to over 70% model compression. These recommendations can be used to inform decisions on implementing quantization techniques in production settings.
